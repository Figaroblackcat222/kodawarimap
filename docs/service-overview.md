# サービス概要

## サービス名

kodawarimap

## 何を作るのか

「他人の評価」ではなく「自分のこだわり」を地図に刻み、資産化するパーソナル地図アプリ。
MapLibreベースのローカルファースト型 趣味記録PWA。

- **技術の核**: MapLibre GL JSを採用。趣味（カテゴリー）に合わせて背景地図のスタイル（登山用グレースケール、夜間自動ダークなど）をダイナミックに切り替える
- **Exif活用**: 写真をアップするだけで撮影場所・日時・カメラ情報を自動プロット。入力負荷を極限まで削減
- **プライバシー設計**: ログイン不要のローカル完結型からスタートし、必要な時だけクラウドと同期する

## ターゲットユーザー

- **趣味人**: 写真家、登山家、釣り人、キャンプ愛好家など「場所」に強い思い入れがある人々
- **プライバシー重視派**: 自分のデータを勝手に収集されたくない、公開範囲を100%コントロールしたい層
- **情報の鮮度を求める人**: 大手地図アプリには載っていないニッチで最新の情報を求める層

## 大切にしたいこと

- **Exifファースト**: 「写真を1枚選ぶだけ」で記録が完了する魔法のような体験
- **専門性の高い地図表現**: カテゴリーに応じて最適な地図（等高線、3D地形、ダークモード等）を提供
- **ユーザー主導権**: ローカル保存を基本とし、ビジネスとしての信頼を「透明性」で築く

## ビジョン（最終目標）

「世界中のニッチな情熱が、最適な地図の上で資産化されている状態」

個人のこだわりが詰まった「自分専用マップ」を束ね、既存の地図サービスでは到達できない、
深いコンテキストを持つ専門地図プラットフォームを構築する。

## 将来的に追加したい機能

- **収益化（課金分岐）**: クラウドバックアップ、Web公開用プロ版、専門地図データの提供
- **趣味別カスタムフォーム**: 趣味ごとに異なる入力項目（魚種・水温・使用レンズ・標高など）
- **プライバシーゾーン**: 自宅・職場など特定範囲を自動で非公開にする機能（設定画面に実装予定枠あり）
- **他人の公開マップを表示**: 他ユーザーが一般公開したピンを地図上に重畳（設定画面に実装予定枠あり）
- **オフラインマップ（エリア保存）**: PMTiles移行後に実装。設定画面に実装予定枠あり
- **テンプレート共有**: ユーザー間で自作の入力フォーム設定をシェア
